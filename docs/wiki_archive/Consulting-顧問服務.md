# 顧問服務 Consulting

**路徑：** `/consulting`

## 功能定位
顧問服務頁是平台商業服務模組的總入口。

## 功能說明
本頁說明五大輔導模組、中控儀表板與加購市場，讓企業可以清楚理解平台可提供的顧問服務內容與導入方式。

## 主要內容
- 五大輔導模組
- 中控儀表板
- 加購市場
- 顧問服務說明

## 使用方式
1. 了解顧問服務模組
2. 選擇需要的方案
3. 連結顧問專區或諮詢頁
4. 進一步導入客製化服務

## 使用場景
- 顧問方案介紹
- 服務模組選購
- 輔導資源展示

## 頁面目標
- 讓服務內容標準化、可理解
- 建立平台商業服務轉化入口

## 解決的痛點
- 顧問服務難標準化
- 企業不清楚可購買哪些模組

## 目前成果
- 已具備顧問服務頁面框架
- 已整合五大模組概念
