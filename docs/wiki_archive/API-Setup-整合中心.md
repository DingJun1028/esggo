# 整合中心 API Setup

**路徑：** `/api-setup`

## 功能定位
整合中心是平台外部系統串接、連接器狀態與環境變數管理頁。

## 功能說明
本頁顯示 API 連接器狀態、Webhook 監控與環境變數校驗結果，協助維持整合層的穩定運作。

## 主要內容
- API 連接器狀態
- Webhook 監控
- 環境變數校驗
- 外部整合健康度

## 使用方式
1. 查看各整合服務狀態
2. 檢查 Webhook 是否正常
3. 驗證環境變數設定
4. 排除串接問題

## 使用場景
- 監控整合狀態
- 檢查系統串接健康度
- 驗證部署設定

## 頁面目標
- 建立整合層可視化管理機制
- 降低外部服務故障排查成本

## 解決的痛點
- 外部整合故障不易即時察覺
- 環境變數錯誤難追查
