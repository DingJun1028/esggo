# 證據金庫 Vault

**路徑：** `/vault`

## 功能定位
證據金庫是平台的佐證文件與可信驗證中心。

## 功能說明
管理所有 ESG 佐證文件，支援 Hash 雜湊值、ZKP 驗證狀態、檔案大小與對應 GRI 條款顯示。

## 主要內容
- 文件上傳與管理
- Hash 值
- ZKP 驗證狀態
- 檔案大小
- 對應 GRI 條款

## 使用方式
1. 上傳佐證文件
2. 指定文件類型與對應條款
3. 檢視 Hash 與驗證狀態
4. 供報告與審核模組引用

## 使用場景
- 文件上傳與管理
- 報告佐證對應
- 外部審計前文件準備

## 頁面目標
- 建立證據集中管理機制
- 讓文件與揭露內容具備可信關聯

## 解決的痛點
- 佐證檔案分散難管理
- 文件與報告內容難以對應
- 缺乏驗證狀態顯示

## 目前成果
- 已具備 Evidence Vault SEED 設計
- 已形成 ZKP 驗證狀態顯示能力
