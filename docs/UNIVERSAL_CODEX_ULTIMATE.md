## 3. 5T 協議門：數據治理矩陣 (The 5T Protocol)

所有數據流必須依序通過五道門徑，實現「淨化與刻印」：

1. **真 (Truth) - Traceable**
   **Origin-verified**: 每一筆數據必須具備可追溯的 `source_origin`，狀態為 **可溯源**。
2. **善 (Goodness) - Transparent**
   **Algorithmically verifiable**: 算法透明，通過 [ISO-14064-1] 驗算標準，狀態為 **可透明**。
3. **美 (Beauty) - Tangible**
   **UI/UX Excellence**: 將數據轉化為「液態玻璃」質感與動態回饋，狀態為 **可感知**。
4. **信 (Trust) - Trustworthy**
   **Cryptographically secured**: 執行 Hash Lock，形成不可篡改的磐石，狀態為 **不可篡改**。
5. **通 (Transferful) - Trackable**
   **Lifecycle-aware**: 透過 Hook 實現全生命週期的即時追蹤，狀態為 **可追蹤**。
