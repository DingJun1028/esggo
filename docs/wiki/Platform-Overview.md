# 平台總覽 Platform Overview
路徑： / | 權限： ALL_USERS | 所屬旅程： 核心概念

1. 模組定位 (Core Purpose)
ESGGO 善向永續 系統是一套聚焦於企業永續治理的「SaaS 操作系統」。它將「資料輸入 → 策略分派 → AI 撰寫 → 稽核驗證 → 報告發佈」串聯為一個閉環旅程。

2. 客戶旅程與 UX 體驗 (Customer Journey & UX)
目標對象與痛點： 企業永續負責人面對零散的數據與複雜的法規，感到無所適從。
體驗高光時刻： 登入後看到清晰的 5T 治理狀態與 AI 建議。

3. UI/UX 視覺與 RWD 基準 (Visual & RWD Baseline)
桌面版佈局： Sovereign Bento Grid.
核心元件： Liquid Glass Cyan 系列元件。

4. 核心邏輯與 5T 協定 (Logic & 5T Protocol)
核心協議： 5T 誠信協議 (Truth, Goodness, Beauty, Trust, Transfer).

5. 功能項目解說和使用技術 (Features & Tech Stack)
全系統採用 Next.js 15 + Supabase + Gemini 2.0.

6. 品質達標與驗收紅線 (QA Red Lines)
🚨 UI 跑版紅線： Z-index 必須分層明確。
