# 🗺️ 客戶旅程地圖 (Customer Journey Map) - WIKI 統一標準指南

> **規範說明**：本文檔基於 **[WIKI 第 66 頁統一標準]** 制定。所有部門在繪製與優化客戶旅程時，必須嚴格遵守此「**以客戶為中心 (Customer-Centric)**」的核心指導原則。

---

## 🌟 核心理念：以客戶為中心

我們不從「公司要賣什麼」出發，而是從「**客戶需要解決什麼問題、經歷什麼感受**」出發。

1. **同理心驅動 (Empathy-Driven)**：感受客戶的痛點，理解他們的真實需求與焦慮。
2. **全通路一致性 (Omnichannel Consistency)**：確保客戶在線上、線下、客服等所有接觸點，都能獲得無縫的優質體驗。
3. **動態演進 (Dynamic Evolution)**：客戶旅程不是靜態的文件，必須根據數據反饋與客戶真實聲音 (VOC) 持續迭代。

---

## 📊 標準化客戶旅程五大階段 (The 5 Stages)

根據統一標準，客戶旅程必須包含以下五個核心階段：

### 1. 認知 (Awareness)
* **客戶狀態**：意識到自己有問題或需求，開始尋找解決方案。
* **客戶思考**：「我該如何解決這個問題？」、「這家公司能幫我嗎？」
* **關鍵接觸點 (Touchpoints)**：社群媒體、SEO 搜尋、數位廣告、口碑推薦。
* **我們的目標**：吸引注意力，建立品牌初步信任。

### 2. 考慮 (Consideration)
* **客戶狀態**：正在評估不同的選項，比較產品功能、價格與評價。
* **客戶思考**：「這個方案真的適合我嗎？」、「跟其他競品比起來哪裡更好？」
* **關鍵接觸點**：官方網站、產品著陸頁 (Landing Page)、試用體驗、線上客服 (Chatbot)。
* **我們的目標**：提供清晰、透明的資訊，降低決策摩擦力。

### 3. 購買/決策 (Purchase/Decision)
* **客戶狀態**：決定購買，正在進行結帳或簽約流程。
* **客戶思考**：「付款過程安全嗎？」、「希望能趕快開始使用。」
* **關鍵接觸點**：購物車、結帳頁面、業務人員、確認信件。
* **我們的目標**：打造極致流暢、無痛的結帳與 onboarding (引導) 體驗。

### 4. 留存/使用 (Retention/Usage)
* **客戶狀態**：正式成為用戶，開始使用產品或服務。
* **客戶思考**：「這東西好難上手...」、「客服能幫我解決問題嗎？」
* **關鍵接觸點**：產品介面 (UI/UX)、幫助中心、售後服務、客戶成功團隊 (CSM)。
* **我們的目標**：確保持續的價值交付，主動關心並快速解決客戶困難。

### 5. 倡導/擁護 (Advocacy)
* **客戶狀態**：對產品非常滿意，願意推薦給他人。
* **客戶思考**：「這太棒了，我要推薦給朋友！」
* **關鍵接觸點**：社群分享、忠誠度計畫、推薦碼機制、用戶社群。
* **我們的目標**：將滿意客戶轉化為品牌大使，建立正向飛輪。

## 🛠️ 地圖繪製必備要素 (Required Elements)

在繪製具體的 Journey Map 時，必須包含以下 **統一標準維度**：

| 維度 (Dimension) | 說明 (Description) | 以客戶為中心的視角 |
| :--- | :--- | :--- |
| **Persona (人物誌)** | 目標客戶的具體畫像 | 「這是真實的人，不是冷冰冰的數據」 |
| **Touchpoints (接觸點)** | 客戶與品牌互動的管道 | 「客戶習慣在哪裡出現，我們就在哪裡」 |
| **Doing (行為)** | 客戶在這個階段正在做什麼 | 「觀察他們的實際行動，而不只是意圖」 |
| **Thinking (想法)** | 客戶內心的疑問與考量 | 「解答他們心中尚未說出口的疑慮」 |
| **Feeling (情緒曲線)** | 高低起伏的情緒狀態 (😀 😐 😡) | 「我們要在他們最挫折時提供溫暖支援」 |
| **Pain Points (痛點)** | 阻礙客戶前進的摩擦力 | 「哪裡讓客戶不爽，哪裡就是我們的第一要務」 |
| **Opportunities (機會點)** | 我們可以改善與創新的地方 | 「如何給客戶創造 "Wow" 的驚喜時刻？」 |

## 📈 衡量指標 (Metrics & KPIs)

為了確保「以客戶為中心」不是空喊口號，我們透過以下指標檢驗各階段成效：

* **NPS (淨推薦值)**：衡量整體客戶忠誠度與推薦意願。
* **CSAT (客戶滿意度)**：衡量特定互動節點的滿意度。
* **CES (客戶費力指數)**：衡量客戶完成某項任務的難易度 (越低越好)。
* **Churn Rate (流失率)**：衡量留存階段的健康度。
