# ESGGO 使用者操作手冊 User Guide 📘
## 版本：v8.5.1 | 狀態：Operational-Ready

歡迎使用 **ESGGO 善向永續** 系統。本手冊旨在引導使用者快速熟悉系統操作，從帳號設定到報告發佈。

---

## 1. 系統登入與設定 (Onboarding)
*   **登入系統**：使用公司配發的帳號登入。
*   **個人檔案**：前往 `/profile` 設定您的職位與聯絡資訊。
*   **API 串接**：技術人員請前往 `/api-setup` 配置 ERP、HRIS 等外部數據源。

## 2. 策略盤點與分派 (Strategy)
*   **重大性評估**：前往 `/materiality` 進行雙重重大性評估，定義年度核心議題。
*   **任務分派**：在 `/tasks` 頁面將特定數據填報任務指派給各部門窗口。

## 3. 數據採集與填報 (Operations)
*   **環境數據**：前往 `/environmental` 填報溫室氣體 (Scope 1-3)、能耗與水資源數據。
*   **社會影響**：前往 `/social` 填報員工福利、教育訓練與職安數據。
*   **交易所整合 (Swap-DeFi)**：在任務中心可執行資產轉換任務，透過 `Swap-DeFi-TEST-UMES-ONLINE` 確保數據真實性。

## 4. AI 協作與撰寫 (AI Copilot)
*   **SustainWrite**：前往 `/editor` 使用 AI 擴寫與潤稿功能。
*   **數位分身**：前往 `/digital-twin` 進行績效模擬與優化建議分析。

## 5. 確信審計與發佈 (Audit)
*   **證據金庫**：所有上傳的文件皆儲存於 `/vault`，並掛載密碼學 Hash 鎖。
*   **VerifyLinkT**：稽核員在 `/audit-verify` 執行零知識證明 (ZKP) 驗證。
*   **一鍵發佈**：確認無誤後，於 `/publish` 生成符合國際標準的 PDF/HTML 報告。
